ELEMENTOS ESTRUTURAIS

[DIV] São caixas organizadoras. Muito utilizado para organizar sites que possuem grandes estruturas (textos, fotos, vídeos, informações etc). São muito utilizados e ajuda muito a estilizar o site no CSS. As [div] são invisíveis.


[span] São parecidos com a div porém para quantidade de textos menores. Também para facilitar a estilização no CSS.

Ex: Em uma linha de um paragrafo você quer que toda essa linha esteja na cor preta e somente uma palavra na cor vermelha. Você utiliza o span para selecionar a palavra em questão.

INPUT


A tag input tem várias opões ou tipos de aplicações

Autocomplete; você inicia o preenchimento e automaticamente o site preenche com suas informações salvas. É preciso dizer qual a informação “rua endereço, e-mail, telefone etc”

Disable; desabilita o input, não pode escrever.

Value; já coloca um valor no input. Ex Lucival

Readonly; não desabilita, mais não permite escrever no input.

Autofocos; já começa o site com o input com foco. Se colocar o autofocos em mais de um input ele só focara no primeiro.

Placeholder; coloca a informação a ser preenchida no input. Quando começa a escrever no input some a informação.


Quando utilizamos a tag [input] dentro da tag [label] o leitor do site (programa) entende que o paragrafo e o input fazem parte da mesma coisa.



Color; coloca um input de cor

Ranger; coloca uma barra com ranger. E temos a opção de colocar o valor mínimo, máximo e começa de um valor pré-definido.


File; para dowload de arquivo

Multiple; dowload de vários arquivos ao mesmo tempo.

Accept; escolhe o tipo de arquivo permitido, se é somente vídeo, fotos etc.

FORMULÁRIOS


A tag [form] serve para criar um formulário e captar várias informações de entrada. Dentro do formulário sempre deve haver um [button] com a especificação submit. Pois ele fará a leitura dos dados digitados e enviara para o lugar onde será guardado ou tratado as informações. Dentro da tag [form] é preciso ter declarado a action para onde será enviado, e qual method será utilizado (Ex: GET, POST)


Digite seu endereço


BUTTON


O uso da tag [button] dentro da tag [form]. Button do tipo Reset reseta tudo que estiver escrito no formulário. Button do tipo submit envia as informações digitadas no formulário para o local definido na ação (site, servidor, tela etc).

O uso do button de forma geral, temos várias opções uma das mais utilizadas é o botão do tipo button, onclick. Com esta opção eu defino qual ação o botão fará após ser clicado. Essas ações definimos no Java escript. Podemos abrir um menu, outro site, comprar algo, fazer um calculo etc.
